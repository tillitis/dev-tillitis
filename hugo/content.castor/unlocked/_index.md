# TKey Unlocked

The unlocked version of the TKey comes with an FPGA without
configuration, i.e., it does not have a bitstream programmed. This
means that you can provision the TKey yourself.

Castor is under development and these pages about how to build
bitstream and program will be updated. Meanwhile there is information
in the [readme](https://github.com/tillitis/tillitis-key1) and
[release
note](https://github.com/tillitis/tillitis-key1/blob/main/doc/release_notes.md#upcoming-release-castor)
for those interested in experimenting with Castor.
